== Description ==

[Beam Analytics](https://beamanalytics.io) is the most affordable GDPR compliant Google Analytics alternative on the market. And it comes with cohort retention and funnel analysis.

With this plugin, you can:

* Easily add the Beam Analytics tracking code to your website
* Select user roles to exclude from tracking
* Select environments to exclude from tracking

**[Upgrade to Premium](https://wpsunshine.com/plugins/beam-analytics/?utm_source=wordpress.org&utm_medium=link&utm_campaign=beam-analytics)**

Easily create *custom conversions/events* in Beam from the plugin settings page without any code:

* *On click* of any element via CSS selector
* *Scroll* to an element via CSS selector
* *Time* on page
* Track across entire site or only on a specific page

*Automatic conversion/event tracking* for many popular plugins:

* WooCommerce
* Easy Digital Downloads
* Sunshine Photo Cart
* Gravity Forms
* WPForms
* WS Form
* Ninja Forms
* Formidable Forms
* Forminator
* Contact Form 7
* LifterLMS
* Paid Memberships Pro
* WP Simple Pay
* Charitable

== Installation ==

1. Upload and activate the plugin
2. Go to Settings > Beam Analytics
3. Get your token and paste it into the settings
